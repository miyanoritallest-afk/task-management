# タスク管理アプリ 要件定義書

作成日：2026-04-24

---

## 1. アプリの目的

Trello風のタスク管理アプリ。  
タスクをカード形式で管理し、進捗状況を視覚的に把握できる。

---

## 2. 画面構成

```
┌─────────────────────────────────────────┐
│  タスクボード（ヘッダー）                │
├───────────┬───────────┬─────────────────┤
│   Todo    │  進行中   │      完了       │
├───────────┼───────────┼─────────────────┤
│ [カード]  │ [カード]  │   [カード]      │
│ [カード]  │           │                 │
│ + 追加    │ + 追加    │   + 追加        │
└───────────┴───────────┴─────────────────┘
```

---

## 3. 機能一覧（フェーズ別）

| フェーズ | 機能 | 内容 |
|---------|------|------|
| 1 | カード追加 | タイトルを入力して追加 |
| 1 | カード削除 | ×ボタンで削除 |
| 1 | カラム間移動 | ボタンで左右に移動 |
| 1 | データ保存 | ブラウザに自動保存（ローカルストレージ） |
| 2 | 説明文 | カードに詳細テキストを追加 |
| 2 | 期限設定 | 締切日を設定・表示 |
| 3 | 並び替え | カラム内の順番を変更 |
| 3 | ドラッグ&ドロップ | カードをドラッグで移動 |
| 4 | 複数ボード | ボードを複数作成・切り替え |
| 4 | カード色付け | カードに色をつける |

---

## 4. データ構造

```
ボード
└── カラム（Todo / 進行中 / 完了）
    └── カード
        ├── ID（識別番号）
        ├── タイトル
        ├── 説明文
        ├── 期限
        └── 色
```

---

## 5. 技術スタック

| 役割 | 技術 |
|------|------|
| フレームワーク | Next.js 14 |
| スタイリング | Tailwind CSS |
| データ保存 | ローカルストレージ |
| ドラッグ&ドロップ | dnd-kit（フェーズ3で導入） |

---

## 6. 開発フェーズ

### フェーズ1（基本機能）
- カードの追加・削除
- カラム間の移動（ボタン操作）
- ローカルストレージへの自動保存

### フェーズ2（カード詳細）
- カードへの説明文追加
- 締切日の設定と表示

### フェーズ3（操作性向上）
- カラム内の並び替え
- ドラッグ&ドロップでの移動

### フェーズ4（拡張機能）
- 複数ボードの作成・切り替え
- カードへの色付け
